## 🚀 How to Run the Projects

### Prerequisites
- Go installed (version 1.21 or higher)
- Git installed

### Running a project
```bash
# Clone the repository
git clone https://github.com/kayquesza/learninGo.git

# Navigate to the project directory
cd learninGo/[project-name]

# Run the project
go run main.go
```
